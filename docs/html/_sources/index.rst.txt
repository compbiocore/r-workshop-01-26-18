Purpose
=======

This site hosts the Jupyter notebook for the CBC Introductory R Workshop.  The notebook will be served on the web as a static HTML page, meaning that while it looks like a live notebook, it cannot be edited.  To obtain an editable notebook file, please click the 'download notebook' link.
